# YSXK-DataFrameArea 数据框区域

YSXK-DataFrameArea 数据框区域 是 YSXK-DataFrame 数据框 带标题和分界线版本，多了一个areaData层级不同，其他API完全相同
所有数据需要传递字符串类型
基于业务需求定制，所有组件默认为position: absolute;定位文档流
